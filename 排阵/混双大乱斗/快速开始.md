# 混双大乱斗 - 快速开始

## 📋 功能说明

实现A组男队员和B组女队员的混双对决，支持**2个场地同时进行**，确保同一轮次球员不重复。

## 🚀 快速使用

### 方式一：使用示例脚本（推荐）

1. 编辑 `run_example.py`，修改球员名单
2. 运行：`python run_example.py`
3. 查看生成的 `混双大乱斗_对阵表.xlsx`

### 方式二：直接运行主程序

```bash
python mixed_doubles_chaos.py
```

## 📊 示例数据

**A组男队员（6人）**：林锋, 王小波, 陈顺星, 罗琴荩, 苏大哲, 黄冬青  
**B组女队员（6人）**：田茜, 李祺祺, 高洁, 滕菲, 崔倩男, 李杏芝

**生成结果**：
- 15轮比赛
- 30场比赛（每轮2场并发）
- 每轮8名球员参与（无重复）

## ✨ 核心特性

✅ **并发进行**：2个场地同时比赛  
✅ **球员不重复**：同轮次每个球员只出现在一个场地  
✅ **合理配对**：每场比赛4人完全不同  
✅ **场次均衡**：自动分配参赛次数  
✅ **Excel导出**：标准格式，可直接打印使用

```python
# 调整参数

```

## 💡 使用提示

1. **最少人数**：至少需要2男2女
2. **每次不同**：由于随机性，每次运行结果会不同
3. **场次分布**：球员参赛场次会有差异（7-12场属于正常范围）
4. **Excel格式**：可直接打印，比分栏留空供现场填写

## ❓ 常见问题

**Q: 如何增加比赛轮数？**  
A: 设置 `max_rounds=20` 或更多

**Q: 可以用3个场地吗？**  
A: 可以，设置 `court_count=3`

**Q: 为什么有些球员场次多？**  
A: 随机算法导致，属正常现象。如需更均衡可增加轮数
