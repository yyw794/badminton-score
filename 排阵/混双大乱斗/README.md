# 混双大乱斗排阵系统

## 功能说明

A组男队员和B组女队员依次组合对战，支持2个场地同时进行比赛。

### 核心特性

✅ **并发进行**：2个场地同时比赛，提高比赛效率  
✅ **球员不重复**：同一轮次中，每个球员只能出现在一个场地  
✅ **合理配对**：确保每场比赛的4名球员都不同（无相同男队员或女队员）  
✅ **场次均衡**：自动分配，让每位球员的参赛次数相对均匀  
✅ **Excel导出**：生成格式规范的对阵表Excel文件

## 使用方法

### 1. 修改球员名单

编辑 `mixed_doubles_chaos.py` 文件中的 `main()` 函数，修改球员名单：

```python

group_b_females = ["田茜", "李祺祺", "高洁", "滕菲", "崔倩男", "李杏芝"]
```

## 高级配置

### 调整最大轮数

在调用 `generate_mixed_doubles_matches()` 时，可以指定 `max_rounds` 参数：

```python
matches = generate_mixed_doubles_matches(
    group_a_males=group_a_males,
    group_b_females=group_b_females,
    court_count=2,
    max_rounds=20  # 自定义最大轮数
)
```

如果不指定，默认会根据人数自动计算（最多15轮）。

### 调整场地数量

```python
matches = generate_mixed_doubles_matches(
    group_a_males=group_a_males,
    group_b_females=group_b_females,
    court_count=3  # 3个场地同时进行
)
```

## 验证规则

系统会自动验证以下规则：

1. ✅ 同一轮次中，每个球员只能出现在一个场地
2. ✅ 每场比赛中，两个配对的男队员不能相同
3. ✅ 每场比赛中，两个配对的女队员不能相同
4. ✅ 不会重复使用相同的对决组合

## Excel格式

生成的Excel文件包含以下列：

| 列名 | 说明 |
|------|------|
| 轮次 | 第几轮比赛 |
| 场地 | 几号场地 |
| 类型 | 比赛类型（混双） |
| 对阵 A (男/女) | A队配对（男队员/女队员） |
| 比分 A | A队比分（留空待填） |
| 比分 B | B队比分（留空待填） |
| 对阵 B (男/女) | B队配对（男队员/女队员） |

## 注意事项

1. **人数要求**：至少需要2名男队员和2名女队员才能生成比赛
2. **随机性**：每次运行结果可能不同（使用了随机打乱）
3. **场次分布**：由于是随机安排，球员的参赛场次会有差异（通常在合理范围内）

## 技术实现

- **语言**：Python 3
- **依赖库**：openpyxl（用于Excel导出）
- **算法**：组合枚举 + 约束满足

## 常见问题

### Q: 为什么有些球员场次多，有些少？
A: 由于随机性和约束条件，场次分布会有一定差异。如果需要更均衡的分配，可以增加轮数或调整算法。

### Q: 可以生成更多轮次吗？
A: 可以，通过设置 `max_rounds` 参数来增加轮数。但要注意，随着轮数增加，找到符合条件的组合会越来越难。

### Q: 支持其他比赛类型吗？
A: 当前版本仅支持混双对决。如需支持其他类型（如男双、女双），可以参考项目中的其他排阵脚本。
