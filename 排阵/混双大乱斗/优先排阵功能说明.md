# 优先排阵功能说明

## 需求背景

崔倩男需要提前回家，因此需要让她的比赛安排在前面的轮次，连续上场不休息。

## 实现方案

### 1. 代码修改

在 `mixed_doubles_chaos.py` 中添加了优先排阵功能：

#### 新增参数
- `priority_players`: 需要优先排阵的球员列表（如需要提前回家的球员）

#### 优先级调整
在公平模式中，增加了第5级优先级（最高优先级）：
```python
# 优先级5【特殊】：包含优先排阵球员（最高优先级）
priority_in_match = match_players & set(priority_players)
can_play_priority = [p for p in priority_in_match if player_games[p] < target_games]
priority += len(can_play_priority) * 50000
```

这个优先级的权重为 **50000**，高于其他所有优先级：
- 优先级2（休息后必须上场）：10000
- 优先级1（场次少的优先）：100
- 优先级3（新配对）：10

### 2. 使用方式

在主函数中调用时传入优先球员列表：

```python
matches = generate_mixed_doubles_matches(
    group_a_males=group_a_males,
    group_b_females=group_b_females,
    court_count=2,
    mode="fair",
    priority_players=["崔倩男"]  # 崔倩男优先排阵
)
```

### 3. 测试结果

**普通模式（无优先排阵）**：
- 崔倩男上场轮次: [2, 3, 5, 7, 8, 10]
- 前6轮是否连续上场: False ❌

**优先排阵模式**：
- 崔倩男上场轮次: [1, 2, 3, 4, 5, 6]
- 前6轮是否连续上场: True ✅

## 扩展用法

如果需要多个球员优先排阵，可以传入多个名字：

```python
priority_players=["崔倩男", "李杏芝"]  # 两人都优先排阵
```

## 注意事项

1. 优先排阵球员的总场次仍然保持均衡（每人6场）
2. 优先排阵只影响上场顺序，不影响总场次分配
3. 系统仍会保证同一轮次球员不重复的核心规则
4. 如果优先球员过多，可能会导致排阵困难或无法满足所有约束
