# 排阵改进说明

## 改进内容

### 1. 混双 vs 男双新格式

**背景**：当女性球员不足 4 人时，无法进行女双比赛，女性只能参加混双（每场需要 2 个女性）。这导致女性球员的场次受限。

**解决方案**：引入"混双 vs 男双"格式：
- **A 队**：混双组合（1 男 + 1 女）
- **B 队**：男双组合（2 男）

**优势**：
- 每场只用 1 个女性（传统混双需要 2 个）
- 同样的女性人数可以支持更多比赛
- 增加女性球员的出场机会

**使用条件**：
- 女性球员 < 4 人
- 自动启用

### 2. 动态调整外援球员限制

**背景**：过去外援球员总是被限制在 5-6 场，即使场地很充足。

**新规则**：根据场地充足程度动态调整：

```python
# 计算场地充足度
courts_per_player = court_count / total_players

# 如果场地充足（3 个场地且每场地服务球员≥0.2）
if courts_per_player >= 0.2 and court_count >= 3:
    return MAX_GAMES_INTERNAL  # 外援与内部球员同等待遇（7 场）
else:
    return MAX_GAMES_GUEST  # 场地紧张时，外援限制 6 场
```

**示例**：
- 13 人 + 3 场地 = 0.23 → 外援可打 7 场 ✅
- 20 人 + 3 场地 = 0.15 → 外援限制 6 场

### 3. 严格执行固定场次约束

**修复问题**：之前有 `fixed_games` 约束的球员（如严勇文、崔倩男）被打 6 场而不是 5 场。

**修复方案**：
- 达到 `fixed_games` 后**禁止参赛**（不享受防止冷场规则例外）
- 有 `fixed_games` 的球员不适用防止冷场规则
- 评分算法中添加巨额惩罚（+10000 分）

## 技术实现

### 1. 混双 vs 男双生成

```python
def generate_mixed_vs_mens_matches(males, females):
    """
    Generate mixed doubles vs men's doubles matches.
    
    - Team A: Mixed doubles (1 male + 1 female)
    - Team B: Men's doubles (2 males)
    """
    for male in mixed_males:
        for female in mixed_females:
            remaining_males = [m for m in males if m != male]
            for mens_pair in combinations(remaining_males, 2):
                match = ((male, female), mens_pair)
                matches.append(match)
```

### 2. 动态外援限制

```python
def get_max_games_for_player(player, court_count, total_players):
    courts_per_player = court_count / total_players
    
    # 场地充足时，外援与内部同等待遇
    if courts_per_player >= 0.2 and court_count >= 3:
        return MAX_GAMES_INTERNAL
    
    # 场地紧张时，外援受限
    if is_guest_player(player):
        return MAX_GAMES_GUEST
    
    return MAX_GAMES_INTERNAL
```

### 3. 排阵优先级调整

当女性不足 4 人时，优先安排混双 vs 男双：

## 建议

### 最佳实践

1. **女性球员≥4 人**：可以进行女双，比赛类型更丰富
2. **女性球员 3 人**：自动启用混双 vs 男双，最大化女性参与
3. **女性球员<3 人**：建议邀请更多女性参加

### 场地配置建议

| 总人数 | 推荐场地 | 外援限制 |
|--------|----------|----------|
| ≤12 人 | 2 场地 | 6 场 |
| 13-15 人 | 3 场地 | 7 场（充足）|
| 16-20 人 | 3 场地 | 6 场 |
| >20 人 | 3 场地 | 5-6 场（紧张）|
