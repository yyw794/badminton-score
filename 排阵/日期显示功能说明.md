# 日期显示功能更新

## 更新内容

已将微信接龙中的活动日期自动解析并显示在 Excel 对阵表中。

## 功能说明

### 支持的日期格式

系统自动识别以下日期格式：

1. **YYYYMMDD** (推荐): `20260323 羽毛球活动`
2. **YYYY-MM-DD**: `2026-03-23 羽毛球活动`
3. **YYYY/MM/DD**: `2026/03/23 羽毛球活动`
4. **月日**: `3 月 23 日羽毛球活动`

### 解析逻辑

从微信接龙文本的前 5 行中自动查找日期信息，按以下优先级匹配：

1. 首先尝试 `YYYYMMDD` 格式（8 位连续数字）
2. 然后尝试 `YYYY-MM-DD` 或 `YYYY/MM/DD` 格式
3. 最后尝试 `月日` 格式

如果未找到日期，则不显示日期信息。

## 技术实现

### 正则表达式

```python
# YYYYMMDD 格式
re.search(r'(\d{4})(\d{2})(\d{2})', line)

# YYYY-MM-DD 或 YYYY/MM/DD 格式
re.search(r'(\d{4})[-/](\d{2})[-/](\d{2})', line)

# 月日格式
re.search(r'(\d{1,2})月 (\d{1,2}) 日', line)
```

### 解析流程

```
1. 读取微信接龙文本
   ↓
2. 遍历前 5 行
   ↓
3. 尝试匹配 YYYYMMDD → 成功 → 返回 "YYYY 年 MM 月 DD 日"
   ↓ 失败
4. 尝试匹配 YYYY-MM-DD/YYYY/MM/DD → 成功 → 返回 "YYYY 年 MM 月 DD 日"
   ↓ 失败
5. 尝试匹配 月日 → 成功 → 返回 "M 月 D 日"
   ↓ 失败
6. 返回空字符串（无日期）
```

## 注意事项

1. **日期行位置**: 日期信息必须在微信接龙的前 5 行内
2. **日期格式**: 推荐使用 `YYYYMMDD` 格式（如 `20260323`），最可靠
3. **中文字符**: 月日格式中的"月"和"日"必须是中文字符
