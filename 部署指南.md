# 🚀 部署到 GitHub Pages 指南

## 方式一：GitHub Pages（推荐，最简单）

### 步骤 1：创建 GitHub 仓库

1. 访问 https://github.com
2. 登录你的 GitHub 账号
3. 点击右上角 **+** → **New repository**
4. 填写仓库信息：
   - Repository name: `badminton-score`（或你喜欢的名字）
   - 选择 **Public**（公开）
   - 不要勾选 "Initialize this repository with a README"
5. 点击 **Create repository**

### 步骤 2：推送代码到 GitHub

在终端执行以下命令（替换为你的 GitHub 用户名）：

```bash
cd /Users/yanyongwen712/Documents/pingan_tech_badminton_team

# 添加远程仓库（替换为你的 GitHub 用户名）
git remote add origin https://github.com/你的用户名/badminton-score.git

# 推送到 GitHub
git branch -M main
git push -u origin main
```

### 步骤 3：启用 GitHub Pages

1. 进入你刚创建的 GitHub 仓库页面
2. 点击 **Settings**（设置）标签
3. 在左侧菜单找到并点击 **Pages**
4. 在 **Build and deployment** 部分：
   - Source: 选择 **Deploy from a branch**
   - Branch: 选择 **main**，文件夹选择 **/(root)**
5. 点击 **Save**

### 步骤 4：等待部署完成

- 等待约 1-3 分钟
- 刷新 Pages 设置页面，会看到顶部显示：
  > Your site is live at `https://你的用户名.github.io/badminton-score/`

### 步骤 5：分享到微信

1. 复制生成的链接，如：
   ```
   https://zhangsan.github.io/badminton-score/
   ```
2. 直接发到微信群
3. 微信里点击链接即可访问

## 方式二：Vercel 部署（自动更新）

### 步骤 1：访问 Vercel

1. 访问 https://vercel.com
2. 点击 **Sign Up**，用 GitHub 账号登录

### 步骤 2：导入仓库

1. 点击 **Add New Project**
2. 选择 **Import Git Repository**
3. 找到你的 `badminton-score` 仓库
4. 点击 **Import**

### 步骤 3：部署

1. 保持默认设置即可
2. 点击 **Deploy**
3. 等待约 1 分钟，部署完成后会获得链接：
   ```
   https://badminton-score.vercel.app
   ```

### 优势

- 每次 push 代码自动更新网站
- 自动 HTTPS
- 全球 CDN 加速

---

## 方式三：Netlify Deploy（拖拽部署）

### 步骤 1：访问 Netlify

1. 访问 https://app.netlify.com
2. 注册/登录账号

### 步骤 2：拖拽部署

1. 点击 **Sites** → **Add new site** → **Deploy manually**
2. 将 `排阵/web` 文件夹拖拽到指定区域
3. 等待上传完成，获得链接：
   ```
   https://random-name.netlify.app
   ```

### 自定义域名（可选）

在 Site settings → Domain management 中可以绑定自定义域名

## 📱 微信访问优化

### 生成二维码

部署完成后，可以用以下工具生成二维码，方便微信扫码访问：

1. 草料二维码：https://cli.im/url
2. 输入你的网站链接
3. 下载二维码图片
4. 发到微信群，大家扫码即可访问

### 微信内打开提示

由于微信对部分外链有限制，如果遇到无法打开：

1. 点击右上角 **...**
2. 选择 **在浏览器打开**

## 🔧 更新网站

修改代码后，推送更新：

```bash
# 修改文件后
git add .
git commit -m "更新说明"
git push
```

- GitHub Pages: 约 1-2 分钟自动更新
- Vercel: 约 30 秒自动更新
- Netlify: 手动重新上传

## ❓ 常见问题

### Q: GitHub Pages 访问慢？
A: 国内访问 GitHub Pages 可能较慢，可考虑 Vercel 或 Netlify

### Q: 如何绑定自定义域名？
A: 
- GitHub Pages: Settings → Pages → Custom domain
- Vercel/Netlify: 在设置中添加域名

### Q: 数据能云端同步吗？
A: 当前版本是纯前端，数据存在本地。如需云端同步，需要添加后端服务（如 Supabase）

### Q: 微信群里打不开怎么办？
A: 提示用户点击右上角「在浏览器打开」
